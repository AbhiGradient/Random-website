Random-website

This is a frontend-only car modification parts website created as a beginner learning project using HTML, CSS, and JavaScript.
The project was built to understand how real-world websites are structured, how multiple pages are linked together, and how basic styling and interactivity work.
This project does not include backend functionality such as authentication, databases, or payments.

#Project Purpose
Learn HTML page structure
Practice CSS styling and layouts
Understand JavaScript basics
Learn Git, GitHub, and GitHub Pages
Build confidence by creating a multi-page website

#Features
Multi-page website navigation
Home page (index.html)
Car-specific pages (example: WRX)
Responsive layout (basic)
Clean folder structure

#Technologies Used
HTML5
CSS3
JavaScript
Git & GitHub
GitHub Pages (for hosting)
